The order I recommend you learn in is: Data Modeling first, then SQL, Python, OLAP, dbt, file formats, Spark, Airflow, SE skills, Kafka, Flink, and AI last. Cloud should be one of the last things you learn — knowing Cloud services without the fundamentals only makes you a Cloud user, not a data engineer.
