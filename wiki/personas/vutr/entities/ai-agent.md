Simply said, an agent is just a Language Model in a loop with the tools it needs to get a job done: Brain (the LLM) + Hands (tools) + Nervous System (orchestration). Agent capability climbs through five levels — Level 0 (LLM alone), Level 1 (LLM + tools), Level 2 (context engineering), Level 3 (multi-agent), and Level 4 (self-evolving).
